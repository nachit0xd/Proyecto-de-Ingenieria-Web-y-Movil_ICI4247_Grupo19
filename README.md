# Proyecto-de-Ingenieria-Web-y-Movil_ICI4247_Grupo19
La aplicación a implementar fomentará las ferias locales mediante la difusión digital, permitiendo a los usuarios descubrir ferias a través de un calendario, geolocalización y notificaciones personalizadas. Además, incluirá perfiles de feriantes con sus productos y oficios, junto con sistemas de interacción.
